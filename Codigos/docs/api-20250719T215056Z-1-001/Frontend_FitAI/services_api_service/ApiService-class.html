ApiService

ApiService class

Fornece métodos estáticos para realizar requisições HTTP à API do FitAI.

Esta classe centraliza a configuração da URL base da API e a lógica para montar os cabeçalhos das requisições, incluindo o token de autenticação. Ela suporta alternar facilmente entre um servidor local (para emulador/web) e um servidor remoto via ngrok (para testes em dispositivos físicos).
